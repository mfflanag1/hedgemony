## Table A.2 — Victory Conditions

| Faction | Victory condition |
|---------|-------------------|
| US | IPs > IPs of everyone else, **and** DPRK does not win |
| NATO/EU | IPs > RU's IPs, **and** RU does not win |
| RU (Russia) | IPs ≥ (US IPs − 5) |
| PRC (China) | IPs ≥ (US IPs − 3), **and** DPRK neither wins nor loses |
| DPRK (North Korea) | IPs > 15, **or** US leaves the Korean Peninsula. **Loses if IPs = 0.** |
| IR (Iran) | IPs > 20 |

Multiple factions can satisfy their conditions simultaneously. Per RAND, victory is
secondary to the learning discussion.

## Turns

Default sessions run **up to 16 turns** (often 5–10 are enough). Each turn ≈ one year.
