# Original Hedgemony — Factions & AORs

> Source: RAND TL301 rulebook. Local study use only — see [NOTICE](NOTICE.md).

## Factions (6 players + White Cell)

| Id | Name | Side | Notes |
|----|------|------|-------|
| `US` | United States | Blue | Central player; free-play; only faction tracking Readiness |
| `NATO_EU` | NATO / EU | Blue | U.S. allies; free-play |
| `RU` | Russia | Red | Card-constrained; signals |
| `PRC` | China | Red | Card-constrained; signals |
| `DPRK` | North Korea | Red | Card-constrained; signals; nuclear track |
| `IR` | Iran | Red | Card-constrained; signals |

White Cell = facilitator role (not a faction); sees all hidden state and adjudicates.

## Areas of Responsibility (AORs)

The board's six combatant-command AORs. The default laydown also references named
sub-areas inside an AOR (shown in parentheses); we keep these as labels but place
forces by AOR.

| Id | Name | Sub-areas used in default laydown |
|----|------|-----------------------------------|
| `NORTHCOM` | U.S. Northern Command | CONUS |
| `EUCOM` | U.S. European Command | RU border |
| `CENTCOM` | U.S. Central Command | Iran, Afghanistan, Iraq |
| `INDOPACOM` | U.S. Indo-Pacific Command | PRC, DPRK |
| `SOUTHCOM` | U.S. Southern Command | — |
| `AFRICOM` | U.S. Africa Command | — (e.g. the "Finding Kony" event) |

## Critical Capabilities tracked

`LRF` (long-range fires), `C4ISR`, `IAMD/BMD` (integrated air & missile defense /
ballistic missile defense), `SOF` (special operations forces), `Nuclear`. Each is a
per-faction Mod Level (see default scenario).
